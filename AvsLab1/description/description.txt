Вариант 215, задача №5, дополнительная функция №16.
Квадратные матрицы с действительными числами.

Обобщённый артефакт: объект, содержащий массив и размерность.

Базовые альтернативы:
1. Обычный двумерный массив.
2. Диагональная (на основе одномерного массива).
3. Нижняя треугольная матрица (одномерный массив с формулой пересчета).

Общая для всех альтернатив переменная: размерность – целое число.

Общая для всех альтернатив функция: вычисление среднего арифметического строки (действительное число).

Дополнительная функция.
Упорядочить элементы контейнера по убыванию используя сортировку методом деления пополам (Binary Insertion).
В качестве ключей для сортировки и других действий используются результаты функции, общей для всех альтернатив.

Команда терминала должна иметь один из следующих видов:
command -f <input file name> <output file name>
command -r <matrix type> <matrix size> <output file name>

Формат входных данных в файлах (определён автором работы).
В первой строке одно из трёх чисел: 0, 1 или 2. Определяет тип матрицы (0 - обычная, 1 - диагональная,
2 - нижнетреугольная).
Во второй строке натуральное число - размерность матрицы.
Если матрица обычная, то далее вводятся значения элементов, разделённые пробелом, строка за строкой.
Диагональная и нижнетреугольная матрицы задаются одной строкой.
При некорректных входных данных ошибки выводятся в консоль.
